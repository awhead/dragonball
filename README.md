# 0.0.0.1
## 프로세스
동작을 입력시<br>
1. 플레이어가 컴퓨터인지 사람인지 확인
    1. 컴퓨터라면 랜덤입력
    2. 사람이라면 스캐너
2. 동작은 '기모으기' '막기' '공격'
3. 기를 모으면 기++;
4. 공격을 하면 기--;
5. 기가 0개면 공격을 할 수 없다.
6. 기를 모으는 동안 공격을 할 경우 체력 감소
7. 체력이 0이 되면 게임 끝
